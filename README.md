# Bluestock Mutual Fund Analytics

## Project Overview

Bluestock Mutual Fund Analytics is a data analytics project that analyzes mutual fund data to understand NAV trends, investor transactions, fund performance, and risk-return behavior. The project includes a complete ETL pipeline to clean, validate, transform, and analyze financial datasets using Python, SQL, and visualization tools.

## Objectives

- Build an ETL pipeline for mutual fund datasets
- Clean and preprocess raw data
- Perform data validation and quality checks
- Store processed data in a structured database
- Analyze NAV trends and fund performance
- Calculate returns and performance metrics
- Generate fund scorecards and analytical insights
- Create dashboards for visualization
